                # 图灵机和不可判定问题

                重要性质

                RE 和递归语言在并集、拼接、星闭包、翻转、交集以及逆同态操作下都是封闭的。

                递归语言在差集、补集操作下封闭，同态操作不封闭。主要原因是只要有 h(x)=ϵ 的映射存在，便会有无数种可能，导致无法停机。

                RE 在同态操作下是封闭的。对差集、补集不封闭，主要原因是 TM M 可能永远也不会停机，所以我们无法确定输入是否在差集中。

                一些定理

                • 如果 L 是递归语言，那么 L― 也是递归语言。
                • 如果 L 和 L― 都是 RE，那么 L 和 L― 都是递归的。
                  • 结合上一条定理得到更强的结论：一个语言 L 可判定，当且仅当它是RE，也是co-RE。

                莱斯（Rice）定理

                RE 语言的每个非平凡性质都是不可判定的

                • 递归语言：
                • RE 非递归：HALT，Ld―，Lu，Lne
                • 非 RE ：HALT―，Ld，Le

                相关信息

                Ld={wi|wi∉L(Mi)}。（对角化语言）

                Lu={(M,w)|w∈L(M)}。（通用语言）

                HALT={(M,x)|TM M halts on input x}

                Le={M|L(M)=∅}

                Lne={M|L(M)≠∅}

                # 复杂度

                • 讨论可判定性时，多带图灵机和单带图灵机等价。
                • 讨论复杂度时，二者不等价，但仍然有如下定理：

                令 t(n) 满足 t(n)≥n。则每一个 O(t(n)) 的多带图灵机都有一个 O(t(n)2) 的等价的单带图灵机。

                # P 和 NP

                相关信息

                TIME(t(n))={L|there exists a TM M that decides L in time O(t(n))}NTIME(t(n))={L|there exists a NTM M that decides L in time O(t(n))}

                定义

                定义 P 或者 多项式时间(polynomial-time) 是一类语言，它们在一个单带、确定性图灵机上是多项式时间可判定的。

                P=⋃k≥1TIME(nk)

                定义 NP 或者 非确定性多项式时间(nondeterminstic polynomial time) 是一类语言，它们在一个单带、非确定性图灵机上是多项式时间可判定的。

                NP=⋃k≥1NTIME(nk)

                提示

                其实NP有一个更容易使用的定义：

                NP={L|L is decided by some poly−time NTM}

                语言 L 属于 NP 当且仅当它可以表达为：

                L={x|∃y,|y|≤|x|k,<x,y>∈R}

                其中， R∈P ，判定 R 的多项式时间图灵机 MR 称为 验证机(verifier) 。y 称为 证书(certificate) 或者 证据(proof) 。这个定理的意思是，如果一个问题是NP的，当且仅当，对于某个具体的解，存在多项式时间的判定方法。

                重点

                如何证明一个问题是 NP 问题：

                • 构造多项式时间的 NTM
                • 使用如上的判定定理
                • 使用规约，将老问题规约到要证明的问题

                如何证明一个问题不可判定：

                • 使用规约，将老问题规约到要证明的问题
                • 假设可判定，使用判定机 H 构造原问题的图灵机，推出矛盾。
                • 莱斯定理

                相关信息

                f:Σ∗→Σ∗ 是 可计算的(computable) ，如果存在一个图灵机 Mf ，使得对于每一个 w∈Σ∗ ， Mf 输入 w 后，将 f(w) 写到它的纸带上，然后停机。

                f:Σ∗→Σ∗ 是 多项式时间可计算的(poly-time computable) ，如果对于某个 g(n)=nO(1) ，存在一个 g(n) 时间的图灵机 Mf ，使得对于每一个 w∈Σ∗ ， Mf 在它的纸带上输出 f(w) 并停机。

                A 可以被映射归约(mapping reducible)为 B ，写作 A≤mB ，如果存在一个可计算的函数 f ，使得对于所有的 w ：

                w∈A⇔f(w)∈B

                f 被称为从 A 到 B 的一个归约。

                称 A 可以 多项式时间归约到(poly-time reduced to) B ，记为 A≤PB ，如果存在一个多项式时间可计算的函数 f ，使得

                ∀w,w∈A⇔f(w)∈B

                定理

                • 如果 A≤PB 且 B∈P ，那么 A∈P 。

                • 如果 A≤mB 且 B 是 RE，那么 A 也是 RE。

                • 如果 A≤mB 且 B 是可判定的，那么 A 也是可判定的。

                # NP 完全理论

                称一个语言 B 是 NP完全的(NP-Complete) ，如果它满足以下两个条件：

                1. B∈NP ；

                2. ∀A∈NP,A≤PB 。

                称 B 是 NP难的(NP-hard) ，如果我们忽略第一个条件。

                定理

                • 如果 B 是NP完全的，且 B∈P ，那么 P=NP 。

                • 如果 B 是NP完全的，且对于 C∈NP ， B≤PC ，那么 C 也是NP完全的。

                • 下面三个命题等价：

                  1. P=NP ；

                  2. 每一个NP完全的语言都在 P 中；

                  3. 某些NP完全的语言在 P 中。
